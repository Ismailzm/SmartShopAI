🛒 SmartShopAI – AI-Powered Shopping Assistant

SmartShopAI is an intelligent web-based shopping assistant that enhances the online shopping experience using Artificial Intelligence. It provides personalized product recommendations based on user preferences, mood selection, and interactions.

The system dynamically updates product suggestions in real-time, making shopping faster, smarter, and more engaging.

🚀 Features
🧠 AI-based product recommendation system
🎯 Mood-based filtering (Relaxing, Adventurous, Work Mode)
⚡ Real-time product updates (no page reload)
🎤 Voice assistant support (optional)
🌐 Responsive and user-friendly interface
🔗 Backend integration for dynamic content

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/SmartShopAI.git
cd SmartShopAI
2️⃣ Backend Setup (FastAPI)
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
3️⃣ Run Frontend
Open index.html in your browser
(or use Live Server in VS Code)
🧠 How It Works
User selects a mode (Relaxing / Adventurous / Work Mode)
Frontend sends request to backend
Backend processes logic and filters products
Updated products are displayed instantly
